# StepView 使用说明书

StepView 是一个用来记录“每一步”的桌面画布工具。你可以把一个目标放到画布上，再把过程里的学习、尝试、休息、游戏、见面、吃饭、灵感和完成时刻都接起来，看见自己真实走过的路。

## 1. 启动应用

### 一键启动

双击项目根目录里的 `start.bat`。

### 手动启动

如果你习惯用命令行，可以在项目根目录执行：

```bash
npm install
npm run desktop
```

## 2. 创建一个目标

有两种方式：

1. 在左侧 `Goal` 输入框里写下目标，点击 `Create`。
2. 在画布空白处右键，选择 `Create goal here`。

创建后，画布上会自动出现：

- `🚀` 起点：代表你从这里开始。
- `🏁` 终点：代表你想抵达的地方。

## 3. 记录过程节点

点击节点旁边的 `+`，可以继续添加新的过程节点。

记录时建议写三件事：

- 标题：比如“学会了 React 状态更新”。
- 细节：比如“今天终于理解为什么不能直接改 state”。
- 时间：用来留下这一步发生的日期。

普通状态下，节点会简洁显示标题和日期；点击节点后，可以展开查看完整备注。

## 4. 记录计划节点

如果某一步还没发生，只是一个打算，可以添加计划节点。

计划节点适合记录：

- 明天要补的作业。
- 下周想看的文档。
- 项目下一版要做的功能。
- 想和某个人一起完成的小事。

计划完成后，可以把它标记为已完成，让计划变成真实走过的一步。

## 5. 标记关键节点

有些节点特别重要，可以标记为关键节点。

关键节点适合记录：

- 第一次跑通项目。
- 第一次收到认可。
- 第一次意识到自己真的进步了。
- 一次重要失败、重要休息或重要转折。

## 6. 添加支线

人生和项目都不一定只有一条直线。你可以从已有节点创建支线。

支线类型包括：

- `🧭 我的分支`：自己的备用路线、临时想法、单独探索。
- `🤝 伙伴支线`：朋友、同学、队友一起参与的部分。
- `💗 心动支线`：和喜欢的人、亲密关系、情绪记忆有关的部分。

支线也可以继续添加节点，也可以接回主线。

## 7. 连接不同目标

如果两个目标之间有关联，可以建立跨任务连接。

例如：

- “学 React”影响了“做 StepView”。
- “健身”改善了“学习效率”。
- “一次聊天”改变了“毕业规划”。

这些连接会让你看到：很多看似分散的事情，其实都在互相影响。

## 8. 使用 Emoji 贴纸

左侧 `Stickers` 里有很多 Emoji。

你可以：

- 把 Emoji 拖到画布上。
- 拖动贴纸调整位置。
- 双击贴纸删除。
- 使用 `Emoji rain` 播放一场小小的庆祝。
- 拖动 `Patterns` 把一组贴纸放到画布上。

贴纸不一定要“有用”，它可以只是记录当时的心情。

## 9. 移动画布和节点

- 拖动画布空白处：移动视角。
- 鼠标滚轮：缩放画布。
- 拖动节点：调整节点位置。
- 点击左侧 `Focus`：回到一个更容易查看的位置。

## 10. 完成目标

当一个目标已经走通，可以点击终点里的完成按钮。

完成后，它会进入 `Wins` 里的完成画廊。

在完成画廊里，你可以：

- 查看这个目标的完整旅程。
- 看一共有多少步、多少里程碑。
- 恢复到画布继续编辑。
- 删除不再需要的记录。

## 11. 查看成就

左侧 `Achievements` 可以打开成就收藏。

当你创建目标、添加里程碑、完成计划、连接任务、使用支线、完成目标时，会逐渐解锁徽章。

成就不是为了催你更卷，而是提醒你：你已经做过很多事情了。

## 12. 保存和数据位置

桌面版会自动把数据保存到本地文件。

你可以点击左侧 `Folder` 打开数据文件所在位置。

如果在浏览器里运行，则数据会保存到浏览器本地存储中。

## 13. 清空画布

左侧保存区域有清空按钮。

使用前请确认自己真的不需要当前数据，因为清空会移除画布上的记录。

## 14. 推荐记录方式

你可以从很小的事情开始：

- 今天看了 20 分钟文档。
- 打了一局游戏，心情好多了。
- 吃了新口味的烤冷面。
- 修掉一个困扰很久的 bug。
- 和朋友聊完后没那么焦虑了。
- 又多理解了一点这个世界。

不需要每天都伟大。只要记录下来，你就能看见自己确实在往前走。
