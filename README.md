# StepView

高端现代风格的进度可视化桌面应用：无边界画布、右键创建目标、里程碑节点、备注展开、完成画布、Emoji 趣味贴纸和桌面本地保存。

## 一键启动

双击项目根目录的 `start.bat`。

脚本会自动安装依赖并打开 Electron 桌面窗口。

## 手动启动

```bash
npm install
npm run desktop
```

## 当前能力

- 左侧“新建目标”可直接生成起点和终点。
- 右击画布任意位置可创建任务终点，并自动生成左侧起点。
- 点击节点旁 `+` 可向右延申里程碑，并填写标题、细节、时间戳。
- 未选中节点只展示标题与 `YYYY/MM/DD`，点击节点展示完整备注。
- 节点和 Emoji 贴纸可自由拖动；双击 Emoji 贴纸可删除。
- 点击终点中的“起点终点已打通”会移入完成画布列表。
- 完成画布支持恢复任务和删除任务。
- 桌面版数据保存到 Electron 用户目录的 `stepview-board.json`。
- 左侧“打开数据文件位置”可定位本地数据文件。
